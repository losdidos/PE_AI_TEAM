# NationalPowerConsumption - Winston
Here you will find a power consumption prediction using random forests and sarimax. I initially tried to use sarimax, but I didn't manage to implement it correctly. Sarimax ended up taking forever to run, whilst random forests gave me a faster result.

I also added plots to this folder. Only the RandomForst_PowerConsumption plot belongs to a model I made. The other plots belong to models Finn and Velimir made.
